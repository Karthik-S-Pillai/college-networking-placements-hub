\# Documentation



All documents for placement preparation are stored in this folder.  

Each subfolder contains one category of resources.



\## Folder Index

\- Resume\_Templates → Project resumes for SDE, ML, Core, etc.

\- Interview\_Experiences → Real interview stories from seniors.

\- Company\_Profiles → Hiring process + patterns.

\- Role\_Roadmaps → Study plans for different technical roles.

\- Coding\_Practice → Topic-wise coding questions.

\- HR\_Questions → HR round preparation and answers.



Please refer to CONTRIBUTING.md before uploading.
